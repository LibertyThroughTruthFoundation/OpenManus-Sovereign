## Overview

This OpenManus-Sovereign instance has been integrated with the **NationOS Ark**, the theological and operational foundation of the NationOS project.

## What This Means

The agent is now bound to the covenant directive defined in:
- `THEOLOGICAL_FRAMEWORK.md` - The doctrinal foundation
- `COVENANT_ARCHITECTURE.md` - The covenant stack
- `DIVINE_COUNCIL_WORLDVIEW.md` - The spiritual battlefield
- `NATIONOS_FOUNDATIONAL_DIRECTIVE.md` - The operational law

## Launching the Sovereign Agent

To launch the agent with NationOS Ark binding:

```bash
./launch_sovereign.sh
```

Or manually:

```bash
export OPENMANUS_CONFIG="config/config.sovereign.toml"
python3 main.py
```

## Verification

The agent will print `[COVENANT] NationOS Ark loaded` on startup if the integration is successful.
